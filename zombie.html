Zombie in the Village

Acerca del Proyecto

"Zombie in the Village" fue el primer proyecto que desarrollé durante el ciclo en la Universidad, donde participé como programador. Se realizó en un entorno colaborativo, aplicando los fundamentos del desarrollo de videojuegos a partir de una idea base que fue evolucionando progresivamente hasta convertirse en un producto completamente funcional.

Informacion del Proyecto

  • Rol: Programador
  • N° de Integrantes: 4
  • Tiempo: 5 meses
  • Motor: Unity

Herramientas Usadas

  • Unity
  • C#
  • Aseprite
  • GitHub

Gameplay Systems Implemented

Player Combat System

Implemented 8-direction movement, melee attack with trident, projectile shooting system and damage feedback mechanics.

Enemy AI System

Developed three enemy types with distinct behaviors, player detection and pursuit system, and difficulty scaling.

UI & Progression System

Dynamic score system, ammo counter, coin collection system, and in-game timer integrated into gameplay loop.
